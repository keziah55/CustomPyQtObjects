# CustomPyQtObjects

A repo of convenience classes for PyQt/PySide objects to make them more pythonic.

The project uses [QtPy](https://pypi.org/project/QtPy/), so will work with either PyQt5 or PySide2.

To install:
```
pip install .
```

- TimerDialog

- GroupWidget
